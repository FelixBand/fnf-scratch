# This is a nearly complete Friday Night Funkin' recreation in Scratch.

I started this project around 4 years ago when I was 16 but never got around to publishing it, so here it is.

I'm actually still very proud of this project to this day, it's very solid and very moddable. Give it a go!

# How do I open the project?

You need to install TurboWarp and open the project's sb3 file with that. TurboWarp is a Scratch mod that makes it infinitely better.
